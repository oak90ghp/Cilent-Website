欢迎使用 Cilent Website

这是一个本地网站托管程序，可以在 localhost 上快速搭建网站。

该程序具有以下特性：

  • ✓ 支持自定义网站端口和下载端口
  • ✓ 自动处理 index.html 映射
  • ✓ 完整的访问日志记录
  • ✓ 支持文件下载服务
  • ✓ 一键启动/停止服务

快速开始

  1. 将你的网页文件放在 index/ 文件夹中
  2. 将需要下载的文件放在 request/download/ 文件夹中
  3. 点击 "启动服务" 按钮
  4. 在浏览器中访问相应的 localhost 地址

访问规则

网站访问示例：

  • 文件路径: index/hosting/help/index.html
  • 访问地址: http://localhost:8000/hosting/help
  • 文件路径: index/hosting/help/pageone.html
  • 访问地址: http://localhost:8000/hosting/help/pageone.html

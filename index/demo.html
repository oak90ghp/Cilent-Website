✨ 功能特性

🌐 双端口架构

网站服务和文件下载分别运行在不同端口，相互独立。

📊 访问日志

完整记录所有访问信息，包括 IP、时间、状态码等。

⚙️ 自定义端口

灵活配置网站和下载端口，避免端口冲突。

🎯 智能路由

自动处理 index.html 映射，支持完整的目录访问。

🛡️ 安全防护

防止路径遍历攻击，确保文件访问安全可控。

💻 GUI 管理

图形化操作界面，一键启动/停止/重启服务。

📍 访问规则

规则 1：自动 index.html 映射

当访问的文件夹中只有 index.html 时，可以省略文件名。

文件位置 浏览器访问地址
index/help/index.html http://localhost:8000/help
index/docs/faq/index.html http://localhost:8000/docs/faq

规则 2：具体文件访问

访问特定的 HTML 文件时，需要指定完整文件名。

文件位置 浏览器访问地址
index/help/pageone.html http://localhost:8000/help/pageone.html
index/docs/guide.html http://localhost:8000/docs/guide.html

规则 3：目录列表

访问不包含 index.html 的文件夹时，将显示目录列表。

🚀 快速开始

第 1 步：启动程序

python main.py

第 2 步：配置端口（可选）

在管理界面中修改网站端口和下载端口。

第 3 步：启动服务

点击 启动服务 按钮。

第 4 步：创建你的网页

在 index/ 文件夹中创建你的 HTML/CSS/JS 文件。

第 5 步：访问网站

在浏览器中打开 http://localhost:8000

❓ 常见问题

Q: 如何修改端口？

在管理界面中修改"网站端口"和"下载端口"的值，然后点击"启动服务"即可。请确保新端口未被其他程序占用。

Q: 如何添加 CSS 和 JavaScript？

将 CSS 和 JavaScript 文件放在 index/ 文件夹中，在 HTML 中正常引用即可：

<link rel="stylesheet" href="/style.css"> <script src="/script.js"></script>

Q: 如何创建多层目录？

可以任意创建子文件夹。例如，访问 index/hosting/help/index.html 可以用 http://localhost:8000/hosting/help 访问。

Q: 如何实现文件下载？

在 HTML 中添加下载链接：

<a href="http://localhost:7000/filename.exe">下载</a>

文件需要放在 request/download/ 文件夹中。

Q: 如何查看访问日志？

在管理界面点击"访问记录"按钮，会显示所有访问信息，包括 IP、时间、状态码、User-Agent 等。

Q: 支持图片、视频等媒体文件吗？

完全支持！将媒体文件放在 index/ 文件夹中，在 HTML 中正常引用：

<img src="/images/photo.jpg" alt="图片"> <video src="/videos/demo.mp4"></video>

🔧 技术信息

支持的文件类型

程序支持所有标准 web 文件类型：

  • HTML/CSS/JavaScript
  • 图片（JPG、PNG、GIF、WebP 等）
  • 媒体（MP4、WebM、MP3、OGG 等）
  • 文档（PDF、ZIP 等）
  • 其他任意文件格式

性能和限制

  • 支持无限数量的页面和文件
  • 单个文件大小无限制
  • 支持并发访问
  • 本地 localhost 运行，不受互联网限制

💡 使用技巧

技巧 1：快速重启

点击"重启服务"按钮可以快速刷新服务，无需手动停止后再启动。

技巧 2：组织文件结构

建议使用如下结构来组织文件：

index/ ├── index.html ├── css/ ├── js/ ├── images/ ├── pages/ └── downloads/

技巧 3：测试响应式设计

使用浏览器的开发者工具（F12）来测试响应式设计。程序支持所有标准的 web 技术。

🆘 故障排除

问题：无法启动服务

常见原因：

  • 端口被占用 → 更换端口
  • 防火墙阻止 → 允许通过防火墙
  • Python 环境问题 → 检查 Python 版本

问题：找不到文件（404）

检查：

  • 文件是否在 index/ 文件夹中
  • 文件路径是否与 URL 相符
  • 文件名是否拼写正确

运行测试诊断

运行以下命令进行诊断：

python test.py
